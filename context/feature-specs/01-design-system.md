Read `AGENTS.md` before implementing this feature.
Install and configure `shadcn/ui` (tailwind UI components) on this project.

add these shadcn components:
- Button
- Card
- Dialog
- Input
- Label
- Modal
- Navbar
- Sidebar
- Tooltip

do not modify the generated `components/ui` files after installation.

also install `lucide-react` for icons.

create `lib/utils.ts` with a reusable `cn()` helper for mergin tailwind classes — this is used by the shadcn components.

ensure all components match the exisiting dark theme in `global.css`.

### check when done 
- all shadcn components are installed
- lucide-react is installed
- `lib/utils.ts` is created with `cn()` helper
- all components match the exisiting dark theme
